# pm-strategy

Strategy framework: pluggable strategy trait, threshold and spread strategies, strategy runner, TOML config, backtesting.

## Running Tests

### Unit tests (no network)

```bash
cargo test -p pm-strategy --lib
```

### Integration tests (no network)

```bash
cargo test -p pm-strategy --test '*'
```

### All tests

```bash
cargo test -p pm-strategy
```
